# Automação de Processos – GitHub e Portfólio

Olá!

Este projeto é uma **automação de processos** em Python que visita meu GitHub e meu site de portfólio, simulando comportamento humano com cliques, scroll suave e pausas aleatórias.  

---

# Funcionalidades

- Abre a aba de **Repositories** no GitHub automaticamente  
- Clica em todos os repositórios da lista  
- Rola a página **suavemente até o final** e volta ao topo  
- Faz **pausas aleatórias** para simular comportamento humano  
- Visita o site [Monteiros-Doces-Gourmet](https://pauloalexandredeoliveira.github.io/Monteiros-Doces-Gourmet/) e faz scroll suave  
- Mantém o navegador aberto pelo tempo configurado  
- Fecha sozinho ao finalizar  

---

# Tecnologias usadas

- Python 3.13  
- Selenium  
- WebDriver Manager para Chrome  
- Bibliotecas padrão: `time`, `random`
